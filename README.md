This project implements Ethernet communication using the User Datagram Protocol (UDP) over Internet Protocol Version 4 (IPv4) entirely in the Programmable Logic (PL) of an FPGA using Verilog HDL. The design demonstrates hardware-based packet generation and transmission through the AXI Ethernet 1G/2.5G Subsystem, eliminating the need for software execution on an embedded processor. All packet construction, protocol header generation, checksum calculation, and data transmission are performed directly in hardware.

The project begins by configuring the AXI Ethernet Subsystem through a custom AXI4-Lite Master, which initializes the Ethernet MAC by enabling the required transmitter and receiver functions and configuring the necessary control registers. Once the subsystem is initialized, custom AXI4-Stream modules generate the transmit control information and packet data required by the Ethernet MAC.

The transmitted packet is constructed by combining the Ethernet header, IPv4 header, UDP header, and application payload into a complete Ethernet frame. The IPv4 header checksum and UDP checksum are calculated in hardware before transmission, while the Ethernet Frame Check Sequence (CRC) is automatically generated by the Ethernet MAC. The completed frame is transmitted through the SFP interface and verified using hardware debugging tools and network packet analyzers.

The project serves as a practical implementation of networking protocols on FPGA hardware and provides an understanding of Ethernet communication, AXI protocols, packet encapsulation, checksum generation, and hardware-based network interface design. The modular architecture allows each protocol layer to be developed, tested, and extended independently, making the design suitable for educational purposes, research, and future high-speed networking applications.

The successful implementation demonstrates that reliable Ethernet communication can be achieved entirely within FPGA programmable logic, providing a low-latency and resource-efficient solution for applications requiring deterministic hardware-based network communication.
